# Google Sans Font Files

To use Google Sans Flex font, please download the font files from:
https://fonts.google.com/specimen/Google+Sans+Flex

## Required Files:
Place these font files in this directory:
- GoogleSans-Regular.woff2 (weight: 400)
- GoogleSans-Medium.woff2 (weight: 500)
- GoogleSans-Bold.woff2 (weight: 700)

Alternatively, you can use Google Fonts CDN by updating the layout.tsx file.
